About SQUEEZER..

What is SQUEEZER ?
SQUEEZER is a brilliant way to retrieve documents, videos, images and any other datas from a web page without any effort. SQUEEZER project intends to be a easy global tool for everyday users to retrieve informations they want.
Why would I use it?

You can get what you want with an easy and a much faster way than normal. Who doesn't want that ?

How can I use it ?

Easy peasy lemon SQUEEZY..


1 - Enter the full URL of page which you want to search.
2 - Select data type
3 - Hit the search button !

Voila !


You can easily select from results and download as zip.
